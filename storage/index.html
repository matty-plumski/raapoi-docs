Storage and quotas¶

Currently users have 2 main storage areas:

  • /nfs/home/USERNAME - This is your Home Directory, each user has a 50 GB quota limit.

  • /nfs/scratch/USERNAME - This is you scratch space, each user has a 5 TB quota limit

To view your current quota and usage use the vuw-quota command, for example:

harrelwe@raapoi-master:~$ vuw-quota 

Quotas for home directory

                       Storage  Usage (GB)  Quota (GB)     % Used 
            /nfs/home/harrelwe        0.10       50.00      0.20%
